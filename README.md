## Overview
**ANSEM ORACLE** is the all-knowing degen advisor for the Ansem ($ANSEM) Solana memecoin. It serves as an AI oracle trained on pure on-chain vibes, memecoin cycles, and the collective wisdom of apes, helping you navigate the chaotic waters of Solana memecoin trading.

## Features
- **Live Stats Tracking:** Real-time metrics for Price, Market Cap, 24H Volume, Liquidity, and Holders (powered by DexScreener).
- **Embedded Charting:** Integrated DexScreener charts to analyze price action.
- **Vibe Check:** Track community sentiment with Diamond Hands, Buying More, and Paper Hands pills.
- **Oracle Chat:** Ask the AI oracle anything about the token, from "Wen moon?" to "Is it a rug?" and receive a purely degen analysis.
- **PnL Card Generator:** Automatically generate customized PnL cards to share your bags and trades on social media.

## Project Structure
- `index.html` - The main entry point for the ANSEM ORACLE app.
- `pnl-card.html` - PnL Card generator tool for sharing gains/losses.
- `server.js` - A lightweight Express/Node.js backend.
- `css/` - Vanilla CSS styles (`global.css`, `oracle.css`, `pnl.css`).
- `js/` - Frontend Javascript logic (`oracle.js`, `pnl.js`).

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the server locally:**
   ```bash
   npm start
   ```

3. **Open the App:**
   Visit `http://localhost:3000` (or the port specified in your server) to consult the Oracle!
